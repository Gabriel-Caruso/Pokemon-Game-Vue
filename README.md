# Pokemon Game
Webapp de un juego creado siguiendo el curso de Vue3 de Fernando Herrera.
- Funcionalidades añadidas:
- Dificultad
- Puntuación
- Continuar y guardar puntuación

## Project setup
```
yarn install
```
